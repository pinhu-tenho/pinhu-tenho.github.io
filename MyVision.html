概要

画像検索カメラは指定した画像で画像、ページ、商品、音楽、地図を検索するアプリです。
検索結果を検索結果画面に表示します。
検索結果のページ、商品、音楽をアプリ内ブラウザで表示します。

アプリ内でGoogle Cloud APIを利用するためAPIキーを取得する必要があります。
APIキーの取得方法はブラウザで「google cloud apiキー 取得方法」などで検索して調べてください。(APIキーの取得方法の記事の例はこちら)
APIキーは無料で取得できます。
無料で利用できる検索回数に制限があります。2025年時点で1ヶ月に約1000回まで検索できます。
制限を超えた場合、利用料金が掛かりますのでご注意ください。2025年時点で検索回数に応じた料金が掛かります。
当アプリでは検索1回に対してウェブ検出とランドマーク検出を適用するため、1ユニットのランドマーク検出と1ユニットのウェブ検出に対して料金が請求されます。
Cloud Vision APIの料金はブラウザで「cloud vision api 料金」などで検索して調べてください。(Cloud Vision APIの料金の記事の例はこちら)
利用料金が掛かるのはCloud Vision APIのみでアプリの利用料金は掛かりません。

機能

ホーム画面です。カメラ、フォトライブラリーから画像を取得できます。

検索結果画面の画像タブです。完全一致画像、部分一致画像、類似画像が見つかった場合、画像を表示します。シェアボタンで選択した画像を共有、保存できます。

検索結果画面のページタブです。画像が一致するページが見つかった場合、ページタブを表示します。

検索結果画面のamazonタブです。最良の推測ラベルが見つかった場合、amazonタブを表示します。

検索結果画面のSpotifyタブです。最良の推測ラベルが見つかった場合、Spotifyタブを表示します。

検索結果画面の地図タブです。ランドマークが見つかった場合、地図タブを表示します。
